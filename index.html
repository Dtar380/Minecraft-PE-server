Bienvenidos a la web oficial de Spain-RP.
Este server es un servidor de RolePlay para minecraft Bedrock edition, la cual es la version que utilizan consolas, dispositivos moviles y minecraft windows 10. Este server de momento esta disponible unicamente para dispositivos moviles y minecraft windows 10.

Este server tien unas reglas a seguir y dispone de un sistema de monedas distinto al normal. Estas dos cosas se explican en el apartado de las reglas.

Tambien podreis encontrar un apartado con los baneos, administradores del servidor, moderadores y los jugadores advertidos.
(abreviatura: BAMA)

Por ultimo tambien hay un apartado en el que se muestra la informacion del servidor como:

  • IP y Puerto
  • horarios.
  • mapa/packs
  • eventos
  • Reglas para seguir el role de tu personaje.
